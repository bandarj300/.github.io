About This Site

Portfolio purpose

This website serves as a portfolio for my GV300 work and related methods projects. It keeps the original homework-based GV300 showcase in place while adding a second project page that demonstrates text analysis using open-ended responses from my own survey dataset.

The structure is meant to be straightforward: visitors can quickly understand the site, then move into either an electoral data case study or a text-analysis case study built from homework data.

The strongest coursework pages do two things at once: they show the code and they show that the author understands why the method matters. That is the standard this site is aiming for.
