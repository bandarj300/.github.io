GV300 Showcase

UK election data used to answer two simple but important questions.

This page preserves my existing GV300 homework showcase while presenting it in a cleaner format. The project uses constituency-level election data from the 2024 UK general election to ask whether all votes are equal and where non-voting is concentrated.

Research Question 01

Are all votes equal?

The first part of the assignment tests whether constituency sizes vary enough to create unequal voter influence. By comparing the smallest and largest constituencies, the project shows that representation is not evenly distributed across the country.

The largest constituency in the dataset contains roughly four times as many valid votes as the smallest, which means voters in smaller constituencies are represented by an MP with a smaller voter base.

Research Question 02

Who is not voting?

The second part focuses on turnout. The analysis identifies where non-voters are concentrated and groups constituencies into turnout bands to compare political participation across England, Scotland, Wales, and Northern Ireland.

This makes the turnout gap easier to see and shows how large constituencies with lower participation can contain substantial numbers of citizens who are registered but absent from the final result.

Data Preparation

Cleaning the raw election data

The workflow starts with candidate-level data from the UK Parliament election results API. The data is cleaned, by-elections are removed, variables are renamed, and the file is aggregated to the constituency level so turnout can be calculated against the registered electorate. 

library(tidyverse)
library(readr)
library(stringr)

uk_csv <- "https://electionresults.parliament.uk/general-elections/6/candidacies.csv"
uk_raw <- readr::read_csv(uk_csv, show_col_types = FALSE)

names(uk_raw) <- names(uk_raw) |>
  tolower() |>
  str_replace_all("\\s+", "_")

uk <- uk_raw |>
  filter(!`election_is_by-election`) |>
  select(
    constituency = constituency_name,
    country = country_name,
    valid_votes = candidate_vote_count,
    electorate = electorate
  ) |>
  group_by(country, constituency) |>
  summarise(
    valid_votes = sum(valid_votes, na.rm = TRUE),
    electorate = max(electorate, na.rm = TRUE),
    .groups = "drop"
  ) |>
  mutate(turnout_registered = 100 * valid_votes / electorate)

Key Findings

What the project concludes

  • Constituency sizes vary substantially, so the weight of a vote is not perfectly equal across seats.
  • Turnout patterns differ by country and by constituency type, which matters for political representation.
  • Descriptive statistics and visualisations can turn a broad political question into a precise empirical argument.

This showcase matters because it turns a familiar democratic principle, one person one vote, into something measurable and visible.
