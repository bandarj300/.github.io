Data Preparation

The dataset used here contains country-level indicators commonly studied in comparative politics: GDP per capita, a democracy score (on a 0–10 scale), population, life expectancy, and geographic region. The code below shows how the data were assembled and prepared for analysis.

Visualisation: Democracy and Prosperity

The figure below plots GDP per capita against democracy score for each country, with points sized by population and coloured by region. This visualisation is the piece of work I am most proud of from the course because it communicates a complex, multi-dimensional relationship in a single, readable figure.

library(ggplot2)

ggplot(country_data, aes(x = democracy_score, y = gdp_per_capita)) +
  geom_point(aes(size = population_millions, colour = region), alpha = 0.75) +
  geom_smooth(method = "lm", se = TRUE, colour = "grey40", linetype = "dashed", linewidth = 0.8) +
  geom_text(aes(label = country), size = 2.8, vjust = -1.0, check_overlap = TRUE) +
  scale_y_continuous(labels = scales::comma_format(prefix = "$")) +
  scale_size_continuous(range = c(3, 15), name = "Population (millions)") +
  scale_colour_brewer(palette = "Set2", name = "Region") +
  labs(
    title = "Democracy Score vs GDP per Capita",
    subtitle = "Point size reflects population; dashed line shows linear trend",
    x = "Democracy Score (0-10)",
    y = "GDP per Capita (USD)",
    caption = "Data: Illustrative country-level dataset for GV300"
  ) +
  theme_minimal(base_size = 13) +
  theme(
    plot.title = element_text(face = "bold", size = 16),
    plot.subtitle = element_text(colour = "grey40"),
    legend.position = "bottom",
    legend.box = "vertical"
  )

Figure 1: Democracy Score vs GDP per Capita across 20 countries. Points are sized by population and coloured by region.

What the figure shows

The scatter plot reveals a positive association between democracy score and GDP per capita across the 20 countries in this sample. Countries with higher democracy scores — such as Norway, Sweden, and Canada — tend to also have higher GDP per capita, while countries with lower democracy scores — such as Nigeria and Egypt — tend to have lower GDP per capita. The dashed trend line confirms this upward relationship, though there is considerable spread around it, which suggests that democracy alone does not determine economic output.

Point size encodes population, making it easy to see that large countries like India and Indonesia have moderate democracy scores but relatively low GDP per capita, while smaller wealthy democracies cluster in the upper right. The colour coding by region highlights geographic patterns: European countries are clustered toward the upper right, while African countries fall toward the lower left. These patterns are consistent with well-known findings in comparative politics about the relationship between regime type and economic development, though the direction of causality remains debated in the literature.

Correlation Analysis

To support the visual impression, a simple correlation test quantifies the linear relationship.

cor_test <- cor.test(country_data$democracy_score, country_data$gdp_per_capita)

The Pearson correlation coefficient is 0.718, which indicates a strong positive linear association. The p-value is well below 0.05, suggesting the relationship is statistically significant in this sample. However, with only 20 observations, these results should be interpreted cautiously and are intended here as a demonstration of method rather than a definitive finding.
